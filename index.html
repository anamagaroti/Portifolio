PGP

Descrição

Este projeto é uma implementação de criptografia e descriptografia baseada no protocolo PGP (Pretty Good Privacy), utilizando Golang. O objetivo principal é garantir a segurança das comunicações através de criptografia assimétrica, permitindo que mensagens sejam protegidas contra interceptações e acessadas apenas pelos destinatários autorizados.

Tecnologias utilizadas

Linguagem: Golang
Biblioteca: golang.org/x/crypto/openpgp (ou outra que tenha utilizado)
Chaves Assimétricas: Geração e gerenciamento de chaves públicas e privadas
Hashing e Assinatura Digital: Para garantir a integridade das mensagens
CLI (Command Line Interface): Interface para facilitar a interação com o sistema

Funcionalidades

Geração de Chaves PGP: O sistema permite a criação de um par de chaves (pública e privada) para cada usuário.
✅ Criptografia de Mensagens: Mensagens são cifradas usando a chave pública do destinatário, garantindo que apenas ele possa ler.
✅ Descriptografia de Mensagens: O destinatário pode descriptografar a mensagem usando sua chave privada.
✅ Assinatura Digital: Possibilidade de assinar mensagens digitalmente para garantir autenticidade e integridade.
✅ Validação da Assinatura: Verificação de que a mensagem recebida foi realmente enviada pelo remetente.

Arquitetura do projeto

keygen.go → Geração de chaves PGP
encrypt.go → Função para criptografar mensagens usando a chave pública
decrypt.go → Função para descriptografar mensagens com a chave privada
sign.go → Assinatura digital de mensagens
verify.go → Verificação da assinatura digital
